# DATOS DE PRODUCTOS AIGOSTAR

Script en Python que, a partir del **EAN** de productos, consulta la **API de Aigostar** para obtener información del producto. Además, analiza imágenes del producto para localizar y leer el **QR de la etiqueta de eficiencia energética**, con el fin de extraer el **código EPREL**.

---

## Requisitos

- Python 3.13+
- Las dependencias están listadas en `requirements.txt`

### Instalación

```bash
pip install -r requirements.txt
